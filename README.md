🧠 Quiz de Programação
📋 Sobre o projeto

Este projeto consiste em uma página web de quiz sobre programação, desenvolvida utilizando HTML. O objetivo é testar conhecimentos básicos de desenvolvimento web por meio de perguntas interativas.

A página apresenta diferentes tipos de perguntas e campos de resposta, permitindo que o usuário pratique e revise conceitos importantes da área de tecnologia.

🚀 Funcionalidades

Perguntas de múltipla escolha

Perguntas com resposta em texto

Campo de senha

Seleção de data

Menu suspenso (dropdown)

Upload de arquivo

Pergunta baseada em imagem

Tabela de pontuação

Formulário de feedback para o usuário

🛠️ Tecnologias utilizadas

HTML5

🎯 Objetivo do projeto

Este projeto foi criado com o objetivo de praticar conceitos básicos de HTML, incluindo:

criação de formulários

uso de tabelas

inserção de imagens

utilização de diferentes tipos de inputs

organização de conteúdo em uma página web

📚 Aprendizados

Durante o desenvolvimento deste projeto foi possível praticar:

Estrutura básica de uma página HTML

Criação de formulários interativos

Uso de diferentes elementos HTML

Organização de conteúdo na página
